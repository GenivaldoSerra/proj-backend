# PRD - Projeto Backend Cloud Native

## Visão Geral

Este projeto tem como objetivo desenvolver uma aplicação backend moderna utilizando Node.js e PostgreSQL, evoluindo gradativamente para uma arquitetura Cloud Native hospedada na AWS.

O projeto foi concebido como um laboratório prático para aplicar conceitos de:

- Desenvolvimento Backend
- DevOps
- Infraestrutura como Código (IaC)
- Containers
- CI/CD
- Observabilidade
- Cloud Computing

Ao final, a aplicação será implantada na AWS utilizando Terraform, Docker, GitHub Actions e Amazon RDS.

---

# Objetivos

- Desenvolver um backend em Node.js
- Implementar boas práticas de arquitetura
- Containerizar a aplicação
- Automatizar testes e builds
- Provisionar infraestrutura na AWS
- Implantar utilizando CI/CD
- Monitorar aplicação e infraestrutura
- Criar documentação completa
